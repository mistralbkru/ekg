      Данный электронный атлас ЭКГ предназначен для отработки навыков по ЭКГ-диагностике кардиологических заболеваний.
      Рекомендуется для практикующих врачей, студентов старших курсов медицинских ВУЗов.
     Замечания по содержанию и обнаруженным ошибкам принимаются автором по адресу - volgogradsmp@mail.ru.
     Автор - Абашин А.А. 2012 г.
